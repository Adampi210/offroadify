# Roadmap

## Phase 0 — Baseline

- Next.js runs locally
- Claude instructions and project docs exist
- Clean baseline committed

## Phase 1 — Minimal MVP

1. Upload and local preview
2. Server-side image-editing route
3. Connect UI to API with loading/errors
4. Download generated result
5. Deploy and smoke-test

## Phase 2 — Better 2D controls

- Neutral/studio background
- Multiple off-road presets
- Individual modification controls
- Color changes
- Better prompts and mask-assisted editing
- Evaluation dataset and quality rubric

## Phase 3 — Custom model

- Define objective and metrics
- Build dataset
- Establish API baseline
- Train/fine-tune custom model
- Optimize inference

## Phase 4 — End-user inference

- Choose browser, desktop, or mobile runtime
- Evaluate WebGPU/ONNX/native approaches
- Quantize/distill and benchmark

## Phase 5 — Multi-view and 3D

- Multi-view consistency
- Camera controls
- 3D reconstruction/generation
- Mesh/texture cleanup
- Viewer and downloadable formats
